# Monad Metrics
A real-time telemetry dashboard on [Monad](https://www.monad.xyz/) to get network metrics on React frontend using [Chainstack](https://chainstack.com/) designed for fast monitoring and clear insights.

---
## Quick Start
**1. Clone the Repository** 
```bash
git clone https://github.com/utkarshiniarora/monad-metrics
cd be
```
**2. Install Dependencies**
```bash
npm install
```

**3. Set up your `.env`** 
```bash
CHAINSTACK_URL=<Your-Chainstack-Endpoint-URL>
```
For a detailed step-by-step guide with code examples and screenshots, read the full blog: [Monad Block Data for Developers: A Practical Deep Dive]()

**4. Run the Project**
```bash
node index.js
```
